## Overview

The Shanling CD80II is a compact desktop all-in-one CD player that combines CD transport, USB DAC, and headphone amplification in a single unit. This modernized evolution of the original CD80 features an upgraded Cirrus Logic CS43198 DAC chip, dual SGM8262 amplifier chips, and supports high-resolution audio up to 384kHz/32-bit PCM and DSD256. The unit includes both single-ended (3.5mm) and balanced (4.4mm) headphone outputs with substantial power delivery of 215mW and 850mW at 32Ω respectively. Built with a unibody aluminum-alloy chassis and HD850 transport mechanism, the CD80II targets users seeking a comprehensive desktop audio solution with modern connectivity including Bluetooth 5.0 with LDAC support and USB DAC functionality.

## Advice

The CD80II suits users seeking comprehensive desktop audio functionality in a single compact unit, particularly those requiring both CD playback and high-resolution digital file capability with quality headphone amplification. Excellent measured performance specifications make it appropriate for critical listening applications with both sensitive IEMs and higher-impedance headphones through balanced output. However, potential buyers should be aware of documented reliability issues affecting optical output and gapless playback functionality that may impact specific use cases. Users primarily focused on digital sources might consider dedicated USB DAC alternatives, while those requiring reliable optical output should investigate these limitations before purchase. The all-in-one approach provides good value for users needing multiple functions, though individuals requiring only specific capabilities might achieve better cost-effectiveness through dedicated components.
